Comparing weather features at different latitudes

Visualizations

On this site you will find a comparision of weather components made between 500 cities across the world. We compared the cities Latitude with 4 different weather components: 1) Windspeed 2) Max Temperature 3) Humidity 4) Cloudiness To create these visualizations, Python was used to extract, clean and plot the data with the help of PYthon's Pandas and Matplotlib libraries. Bootstrap was used to create this site and share my findings.
